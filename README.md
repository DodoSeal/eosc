# EOSC (Eos Osc)

Eosc is an NPM package that communicates with [ETC Eos](https://etcconnect.com) lighting consoles.

## Installation

Use the package manager [npm](https://npmjs.com) to install eosc.

```bash
npm install @dodoseal/eosc
```
